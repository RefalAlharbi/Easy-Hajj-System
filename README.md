# Easy Hajj System

## About the Project
Easy Hajj System is an AI-powered crowd monitoring and emergency detection system designed to improve safety during Hajj seasons.

The system analyzes live surveillance video streams from surveillance cameras to detect:
- Overcrowding
- Falling incidents
- Fainting cases
- Dangerous crowd situations

It sends real-time alerts containing incident type, location, and visual evidence to the responsible authorities through a centralized dashboard.

## Main Features
- AI-powered real-time detection
- Live monitoring dashboard
- Emergency alert system
- Predictive analytics for high-risk areas
- Analytics and reporting
- Mobile-responsive interface
- Centralized incident management

## Technologies Used
- Python
- OpenCV
- YOLOv8
- TensorFlow
- MySQL
- Visual Studio Code

## Project Objectives
- Improve pilgrim safety during Hajj
- Reduce emergency response time
- Support authorities with smart monitoring
- Enable proactive crowd management using AI
